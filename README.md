# light-bulb-problem
Clone the light bulb problem files into Linux directory.
Execute the script ./pywrap.ksh
Pass the dimmer values as input 
Sample Input Example-
1544206562	TurnOff
1544206563	Delta +0.5
1544210163	Delta -0.25
1544211963	Delta +0.75
1544211963	Delta +0.75
1544213763	TurnOff
EOF
